# Abyssal Portfolio

Desktop portfolio scaffold built with Tauri v2, React, Vite, Tailwind CSS, Framer Motion, and Lucide React.

## Run

1. Install dependencies with `npm install`.
2. Start the frontend dev server with `npm run dev`.
3. Start the full Tauri app with `npm run tauri dev`.

## Stack

- React UI in `src/`
- Tailwind theme in `tailwind.config.js`
- Tauri window config in `src-tauri/tauri.conf.json`
